项目二：车管家消息采集服务器 (Netty + Redis + MySQL)

项目介绍：（2016.09 → 2016.12）
采集宁波市（约 2K 辆）的装有 OBD 盒子的车辆信息，采用 TCP 自定义协议开发。项目基于 Netty 并发框架并使用 Redis 进行数据缓存

项目职责： 参与并协作开发

  1. 根据已有协议文档，设计并实现协议以及相关类（编解码、业务处理类、加解密）
  2. 设计与实现 Redis 的存储结构（业务数据的缓存）
  3. 基于 Netty 模拟客户端，并实现心跳机制和登陆请求
  4. 使用开源轻量级 HTTP 服务器 Nanohttpd 框架，为后台管理提供 OBD 盒子管理功能（默认配置文件的修改、盒子激活等业务）

项目三：微信在线报名 (Struts2 + Tomcat + JSP + MySQL)

项目介绍：（2016.01 → 2016.08）
为宁波市范围内的中小学生（用户约 5百）提供竞赛的微信在线报名平台。用户关注公众号后，在微信浏览器上展示竞赛信息，用户填写相关信息，提交并缴费，完成在线报名

项目职责： 负责整个项目的设计与实现

  1. 前端采用 WeUI、AmazeUI 作为前后端基本页面样式
  2. 视图层采用 Struts2 作为前端控制器替代 Servlet
  3. 业务逻辑层采用基于 ThreadLocal 的自定义类来完成事务管理
  4. 数据访问层采用 Apache DBUtils 框架替代 JDBC 简化开发
  5. 在线缴费采用微信公众号支付

常用的Eclipse快捷键

1. 打开文件列表

Ctrl + E

2. 关闭当前文件

Ctrl + W

3. 关闭所有文件

Ctrl + Shift + W

4. 逐个文件跳跃

Ctrl + PgUp / PgOn

5. 选取Java文件中的内容

Alt + Shift + →

6. 自动生成 get/set、toString、hash方法

Alt + Shift + S

7. 视图切换

Ctrl + F7

8. Quick Access

Ctrl + 3

9. 变量名和方法之间的跳跃

Ctrl + Shift + ↓

10. 修改名字（变量、方法）

Alt + Shift + S

11. 面包屑导航

Alt + Shift + B

12. 显示调用某个方法的所有类

Ctrl + Shift + G

13. 字母小写 / 大写

Ctrl + Shift + Y / X

14. 根据左大括号找到右大括号

Ctrl + Shift + P

15. 从其他试图跳转到代码编辑视图

F12

16. 跳转到已知行数

Ctrl + L

17. 显示 HTML / XML 元素的所有属性

Ctrl + Shift + 逗号

18. 添加BookMark

19. 使用工作集 working sets 对工程进行归类

20. 统一 Eclipse 工作空间的字符编码

windows → preferences → workspace

21. 打开方法调用层次

基于工程的：Ctrl + Alt + H
基于工作空间的：Ctrl + Shift + G

22. 连接终端设备 Linux / 本机（windows）

Ctrl + Shift + Alt + t

23. 在多个 console 进行切换

Alt + F7

24. 下一个错误

Ctrl + .

25. 快速修改

Ctrl + 1

26. 在导航历史记录（Navigation History）中后退

Alt + ←

27. 在导航历史记录（Navigation History）中前进

Alt + →

28. 由函数内部返回调用处（调试）

F7

29. 一直执行到下一个断点（调试）

F8

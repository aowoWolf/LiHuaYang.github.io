    常用的 Eclipse 快捷键

    1. 打开文件列表：Ctrl + E
    2. 关闭当前文件：Ctrl + W
    3. 关闭所有文件：Ctrl + Shift + W
    4. 逐个文件跳跃：Ctrl + PgUp / PgOn
    5. 选取Java文件中的内容：Alt + Shift + →
    6. 自动生成 get/set、toString、hash方法：Alt + Shift + S
    7. 视图切换：Ctrl + F7
    8. Quick Access：Ctrl + 3
    9. 变量名和方法之间的跳跃：Ctrl + Shift + ↓
    10. 修改名字（变量、方法）：Alt + Shift + S
    11. 面包屑导航：Alt + Shift + B
    12. 显示调用某个方法的所有类：Ctrl + Shift + G
    13. 字母小写 / 大写：Ctrl + Shift + Y / X
    14. 根据左大括号找到右大括号：Ctrl + Shift + P
    15. 从其他试图跳转到代码编辑视图：F12
    16. 跳转到已知行数：Ctrl + L
    17. 显示 HTML / XML 元素的所有属性：Ctrl + Shift + 逗号
    18. 添加BookMark：
    19. 使用工作集 working sets 对工程进行归类：
    20. 统一 Eclipse 工作空间的字符编码：windows → preferences → workspace
    21. 打开方法调用层次：

    • 基于工程的：Ctrl + Alt + H
    • 基于工作空间的：Ctrl + Shift + G

    22. 连接终端设备 Linux / 本机（windows）：Ctrl + Shift + Alt + t
    23. 在多个 console 进行切换：Alt + F7
    24. 下一个错误：Ctrl + .
    25. 快速修改：Ctrl + 1
    26. 在导航历史记录（Navigation History）中后退：Alt + ←
    27. 在导航历史记录（Navigation History）中前进：Alt + →
    28. 由函数内部返回调用处（调试）：F7
    29. 一直执行到下一个断点（调试）：F8
    30. 打开资源查找：Ctrl + Shift + R

    常用的 Chrome 快捷键

    1. 打开超链接

    • 按住 Ctrl + Shift 点击页面上的超链接，在新的标签页打开，同时切换到新打开的标签
    • 按住 Ctrl 点击页面上的超链接，在新的标签页打开，不切换到新打开的标签

    2. 标签之间的切换，Ctrl + Pg Up / Pg On

    3. 关闭当前标签页，Ctrl + W

    4. 光标定位到地址栏，Ctrl + L / Alt + D

    • 输入文字后，按 Alt + Enter，在新的标签页上显示搜索内容
    • 输入文字后，按 Ctrl + Enter，在文字前后加上 www. 和 .com

      5. 打开下载 Ctrl + J

      6. 打开隐身窗口，Ctrl + Shift + N

    常用的 Idea 快捷键

    代码自动排版：Ctrl + Alt + L
    复制一行或选中的内容：Ctrl + C
    查找 java 类：Ctrl + N
    查找文件：Ctrl + Shift + N
    查找文件中包含的内容：Ctrl + Shift + F
    代码窗口横向滚动：Shift + 鼠标滚轮
    显示类继承结构：Ctrl + H
    显示类继承结构图（UML）
    选中类名 - 鼠标右键 Diagram - 鼠标右键 Show Implementations - 选中添加子类
    快捷方法显示类继承结构图
    选中类名后 Ctrl + H - 弹出视图中 Ctrl + A - 鼠标右键 Diagram
    显示 javadoc：Ctrl + Q
    重写父类或者接口的方法：Ctrl + O
    删除类中未使用的包路径：Ctrl + Alt + O
    回到上一次光标所在位置：Alt + Ctrl + 左右方向键

    设置鼠标划过显示 javadoc，File -> Setting -> Editor -> General -> Show quick documentation on mouse move
    设置代码默认不折叠，File -> Setting -> Editor -> General -> Code Folding One line method

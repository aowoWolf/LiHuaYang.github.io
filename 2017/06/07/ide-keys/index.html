记录自己在日常开发中，使用各种 IDE、编辑器时，积累的一些快捷键和小技巧，如 Eclipse、Sublime、Chrome、Idea等，会一直更新

Eclipse

快捷键 解释
打开文件列表 Ctrl + E
关闭当前文件 Ctrl + W
关闭所有文件 Ctrl + Shift + W
逐个文件跳跃 Ctrl + PgUp / PgOn
自动生成方法 Alt + Shift + S
视图切换 Ctrl + F7
Quick Access Ctrl + 3
快速导航 Ctrl + 1
变量名、方法跳跃 Ctrl + Shift + ↓
面包屑导航 Alt + Shift + B
显示调用某个方法的所有类 Ctrl + Shift + G
大小写切换 Ctrl + Shift + Y / X
左大括号找到右大括号 Ctrl + Shift + P
从其他视图跳转到代码编辑窗 F12
跳转到已知行数 Ctrl + L
打开方法调用层次 Ctrl + Alt + H \ Ctrl + Shift + G
下一个错误 Ctrl + .
在导航历史记录（Navigation History）中后退 Alt + ←
在导航历史记录（Navigation History）中前进 Alt + →
由函数内部返回调用处（调试） F7
一直执行到下一个断点（调试） F8
打开资源查找 Ctrl + Shift + R
打开类型查找 Ctrl + Shift + T

常用设置和习惯：

  • 在重要的代码块左侧添加 BookMark，能够快速定位到
  • 善用 TODO、XXXX 等代码标记
  • 使用工作集 working sets 对工程进行归类：
  • 统一 Eclipse 工作空间的字符编码：windows → preferences → workspace
  • 连接终端设备 Linux / 本机（windows）：Ctrl + Shift + Alt + t

Sublime Text3

使用 Ctrl + ` 安装 Package Install 的代码，并使用 Ctrl + Shift + P 呼出

删除当前行：Ctrl + Shift + K，使用 Ctrl + D，进行快捷键替换（Preference -> Key Bindings）

常用设置：

安装输入法跟随插件：IMESupport

打开多行编辑模式：Ctrl + Shift + L

Chrome

快捷键 解释
标签之间的切换 Ctrl + Pg Up / Pg On
关闭当前标签页 Ctrl + W
光标定位到地址栏 Ctrl + L / Alt + D
打开下载 Ctrl + J
打开隐身窗口 Ctrl + Shift + N
在文字前后加上 www. 和 .com 输入文字后，Ctrl + Enter
在新的标签页上显示搜索内容 输入文字后，Alt + Enter
打开超链接1 按住 Ctrl + Shift 点击，新的标签页打开，同时切换到新打开的标签
打开超链接2 按住 Ctrl 点击，在新的标签页打开，不切换到新打开的标签

IntelliJ IDEA

常见快捷键

快捷键 解释
代码自动排版 Ctrl + Alt + L
复制一行或选中的内容 Ctrl + C
查找 Java 类 Ctrl + N
查找资源（包括 Java 类） Ctrl + Shift + N
全局查找字符串 Ctrl + Shift + F
窗口横向滚动 Shift + 鼠标滚轮
自动补全语句分号 Ctrl + Shift + Enter
显示类继承结构 Ctrl + H
显示方法\ 类的 JavaDoc Ctrl + Q
重写父类或者接口的方法 Ctrl + O
整理 import Ctrl + Alt + O
回到上一次光标所在位置 Alt + Ctrl + → / Alt + Ctrl + ←
文件切换 Alt + → / Alt + ←
列出所有方法 Alt + 7（Ctrl + F12 更好，可支持查找）
生成方法（getter、setter） Alt + Insert
跳转到当前方法的调用方法 Ctrl + 鼠标左键
跳转到指定行 Ctrl + G
搜索所有资源 double Shift（输入 / 可以支持查找文件夹）
查看一个接口的实现方法 Ctrl + Alt + B
重命名 Shift + F6
替换 Ctrl + R

常用设置

  • 显示类继承结构图（UML）

选中类名 - 鼠标右键 Diagram - 鼠标右键 Show Implementations - 选中添加子

  • 快捷方法显示类继承结构图

选中类名后 Ctrl + H - 弹出视图中 Ctrl + A - 鼠标右键 Diagram

  • 设置鼠标划过显示 javadoc

File -> Setting -> Editor -> General -> Show quick documentation on mouse move

  • 设置代码默认不折叠

File -> Setting -> Editor -> General -> Code Folding One line method

  • 隐藏 .idea 文件夹和 *.iml 文件夹

setting - File Type

1. Servlet 的生命周期？

servlet 生命周期由容器管理，容器启动时加载，容器关闭时死亡

2. 如何解决中文乱码问题？

通过 HttpServletRequest 对象？方法，和 HttpSerlvetResponse 对象的？方法解决中文乱码问题

3. JSP 九大隐式对象？

request、response、session、page、application、error、out、pageContext

4. pageContext 内置对象可以操作哪些域对象中的数据

request 域
application 域
session 域
page 域

四大域：1. ServletContext域 2. Request 域 3. Session 域 4. PageContext 域

6. 简述 MVC

MVC：Model（模型） + View（视图） + Control（控制器）

7. EL 表达式的作用

9. JDBC 的编码步骤

JDBC 编码步骤：
1、注册驱动
2、获取与数据库的链接
3、创建代表SQL语句的对象
4、执行SQL语句
5、如果是查询语句，需要遍历结果集
6、释放占用的资源
